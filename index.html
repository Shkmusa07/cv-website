What I've built

Projects

Forest Fire Early Detection System
INSPIRE MANAK Govt. Funded

A wireless sensor network that detects forest fires in remote areas before they spread. Each node uses temperature and smoke sensors paired with a LoRa radio module, allowing alerts to travel kilometers without any internet or cellular network.

  • Works without internet — uses LoRa long-range radio (up to 10 km)
  • Mesh network — nodes relay alerts across the forest
  • Solar powered — designed for unattended deployment
  • Weighted risk algorithm — reduces false alarms
ESP32 LoRa / SX1278 C++ / Arduino Sensor Networks Solar Power GPS Mapping OV2640 Camera
Live Node Dashboard — Web Interface
In Progress

A real-time web dashboard that shows the GPS location of each fire detection node on a live map, with alert history and battery/signal status for each node.
